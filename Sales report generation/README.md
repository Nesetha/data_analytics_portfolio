# Sales Report Automation using Python & Excel

## Project Overview
Developed an automated sales reporting workflow using Python, Excel, and Pivot-based dashboards. The project cleans raw retail data, validates records, generates KPI reports, and creates business visualizations for decision-making.

## Features
- Data preprocessing and validation
- Duplicate and missing value handling
- Automated KPI generation
- Excel report export
- Pivot table analysis
- Interactive dashboard creation

## KPIs Generated
- Total Orders
- Total Sales
- Average Sales
- Highest Single Sale
- Top Performing Region
- Top Selling Category

## Tools Used
- Python
- Pandas
- Excel
- Pivot Tables
- Data Visualization

## Business Insights
- West region generated highest sales
- Technology category performed strongest
- Product-level analysis identified top-performing subcategories
